# shell 界面相关杂项


## 快捷键
|按键|作用|
|:--|:--|
|`Ctrl + A`|光标移到命令行头|
|`Ctrl + E`|光标移到命令行尾|
|`Ctrl + W`|删除光标处的单词|
|`Ctrl + U`|删除光标前的所有字符|
|`Ctrl + C`|中断当前输入|
|`Ctrl + D`|退出当前终端 ( logout / exit )|
|`Ctrl + L`|清屏 (clear)|


## 系统运行级别
文件位置：/etc/inittab

|级别|权限|备注|
|:--|:--|:--|
|0|所有进程终止 (关机)||
|1|单用户模式 (root 用户系统维护)|所有服务都不会启动|
|2|多用户模式|网络文件系统 NFS 没有启动|
|3|完全多用户模式|标准级别，有 NFS，命令行|
|4||未使用|
|5|GUI 模式|标准级别，有 GUI|
|6|系统关闭重启||
