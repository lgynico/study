# less

分页显示文件
格式：`less [options] <file>`

## 常见用法
```bash
# 显示文件
less <file>

# 分页显示文件
less -S <file>

# 显示文件加行号
less -N <file>
```

## 注意事项
|按键|作用|
|:--|:--|
|`Up`/`Down`|上下移动|
|`PgUp`/`PgDn`|上一页/下一页|
|`q`|退出|
|`G`|移动到文件末尾|
|`gg`|移动到文件开头|

|输入|作用|
|:--|:--|
|`/str`|搜索字符串 str (N/n 上下查找)|
|`:n`|定位到第 n 行|
