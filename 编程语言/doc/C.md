# C
## 基础
### 数据类型
```c
// 整型
[signed|unsigned] char c = '0';
[signed|unsigned] short s = 0;
[signed|unsigned] int i = 0;
[signed|unsigned] long l = 0L; // 0UL
[signed|unsigned] long long ll = 0LL; // 0ULL

// 浮点数
float f = 1.0f;
double d = 1.0; // 1.0d

// 布尔值
bool b = true;

// 字符串
char[] s;
char *s;

// 数组
int arr[];
int arr[][];
```
